###### PhD Thesis Format

Please refer to **_Section 9_** of the [guideline.](https://www.dcu.ie/sites/default/files/2021-09/dcu-postgraduate-academic-regulations_21_22_final_approved-by-academic-council-june-2021_1.pdf)
